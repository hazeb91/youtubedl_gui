============
Installation
============


Stable release
--------------

To install YoutubeDL GUI, run this command in your terminal:

.. code-block:: console

    $ pip install youtubedl_gui

This is the preferred method to install YoutubeDL GUI, as it will always install the most recent stable release.

If you don't have `pip`_ installed, this `Python installation guide`_ can guide
you through the process.

.. _pip: https://pip.pypa.io
.. _Python installation guide: http://docs.python-guide.org/en/latest/starting/installation/


From sources
------------

The sources for YoutubeDL GUI can be downloaded from the `Github repo`_.

You can either clone the public repository:

.. code-block:: console

    $ git clone git://github.com/hazeb91/youtubedl_gui

Or download the `tarball`_:

.. code-block:: console

    $ curl  -OL https://github.com/hazeb91/youtubedl_gui/tarball/master

Once you have a copy of the source, you can install it with:

.. code-block:: console

    $ python setup.py install


.. _Github repo: https://github.com/hazeb91/youtubedl_gui
.. _tarball: https://github.com/hazeb91/youtubedl_gui/tarball/master
